# MARLO-FlutterTask

This is a Flutter application that demonstrates the implementation of  Home screen, Transaction List Screen, along with a Filter Bottom sheet. The app uses the following packages:

- provider: ^6.0.5
- http: ^1.1.0
- shimmer: ^3.0.0
